### Monthly Website Analytics Monthly Comparison

This report shows how important website engagement metrics changed from the previous month with bar charts and a percent change calculation. They also list the top five pages according to the three top website engagement metrics: views, unique users, and average engagegment time for each user.

### Website Analytics Dashboard

This dashboard was made to visualize changes in monthly website engagement metrics along side key events that may have influence on website engagement. 
Key events are plotted in the timeline above the line chart. The date of the events aligns with the monthly website engagement metrics, allowing users to 
easily connect changes in website engagement with events around the same timeframe. 

Clicking on either an event point on the timeline or a month on the graph or table will filter the entire dashboard for that timeframe. 

Additionally, the filters on the top of the page can be used to filter by website page and date, affecting both the table and line chart. The filter for measure names 
along side each graph allow the user the filter by page views, number of unique users, and average website engagement time per user.
